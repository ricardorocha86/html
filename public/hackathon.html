Os 3 Desafios Reais

Use Python, JavaScript e LLMs para resolver problemas reais do turismo

1
Recomendação Inteligente de Destinos
Criar um sistema de IA que recomenda destinos turísticos personalizados baseado em preferências dos usuários, histórico e tendências de mercado.
LLMs Python API RAG
2
Chatbot Assistente de Viagem 24/7
Desenvolver um agente conversacional com IA que responda dúvidas sobre destinos, custos, documentação, e personalize sugestões em tempo real.
Chatbot JavaScript Agents Web
3
Análise Preditiva de Demanda
Criar um modelo que prevê demanda turística por destino, período e perfil de cliente para auxiliar decisões estratégicas de negócio.
Previsão Análise Python Dados

Requisitos & Regras

O que você precisa saber para participar

👥

Equipes

Formadas por 4-5 pessoas. Inscrições individuais aceitas - ajudaremos na formação de equipes!

💾

Entrega Obrigatória

Link do repositório Git com todo o código. Sem código = sem apresentação

⏱️

Tempo de Apresentação

3min30s a 4min30s. Acima de 5 minutos, apresentação interrompida com penalidade

🎓

Público-Alvo

Estudantes universitários de Salvador, cursos de exatas e tecnologia, entusiastas de IA

🏫

Presença Mínima

Pelo menos 2 membros da equipe no dia da apresentação presencial

✅

Validação

Premiação após validação técnica do código e funcionalidade pela organização
